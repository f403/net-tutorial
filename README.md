# Hands-on: Netzwerke für Anfänger

This repository contains the participant materials for the Tübix 2026 tutorial
**Hands-on: Netzwerke für Anfänger**.

- Event: Tübix 2026
- Time/place: 04.07.2026 10:00-11:50, W1 (C111)
- Language: Deutsch / Denglish
- Session page: <https://cfp.tuebix.org/tuebix-2026/talk/XAZGKA/>

By the time you read this README, the theory slides should already have been
introduced. Use this repository to get into the lab environment and follow the
exercises.

## Start here

1. Open <http://tutorial-ui.403.name> in your browser.
2. Log in to Apache Guacamole and open your assigned server group, such as
   `server01`.
3. Open the lab group requested by the exercise, for example `lab00`, `lab01`,
   or `lab02`.
4. Open the node shells requested by the exercise, such as `lab03 - node1` or
   `lab03 - router1`.
5. Follow the hands-on exercises in
   [docs/hands-on-exercises.md](docs/hands-on-exercises.md).

## Reference material

- [docs/theory-slides-ipv4-networking.pdf](docs/theory-slides-ipv4-networking.pdf)
  contains the theory slides in case you want to check them again later.
- [lab-checkpoint-templates/](lab-checkpoint-templates/) contains the
  Containerlab templates used to define the lab checkpoints. You do not need
  these files for the main tutorial, but they are useful if you are curious
  about how the labs are assembled.

## Lab checkpoints

The lab groups `lab00` through `lab06` are checkpoints. Each exercise starts
from a known lab state and moves toward the next checkpoint. If you get stuck,
your tutor may ask you to close the current node shells and open a later
checkpoint lab group so you can continue from a clean state.

# Deutsche Version

Dieses Repository enthält die Unterlagen für Teilnehmende des Tübix-2026-
Tutorials **Hands-on: Netzwerke für Anfänger**.

- Veranstaltung: Tübix 2026
- Zeit/Raum: 04.07.2026 10:00-11:50, W1 (C111)
- Sprache: Deutsch / Denglish

Wenn du diese README liest, sollten die Theorie-Slides bereits vorgestellt
worden sein. Nutze dieses Repository, um in die Lab-Umgebung zu kommen und die
Übungen zu bearbeiten.

### Start

1. Öffne <http://tutorial-ui.403.name> in deinem Browser.
2. Melde dich in Apache Guacamole an und öffne deine zugewiesene Servergruppe,
   zum Beispiel `server01`.
3. Öffne die Lab-Gruppe, die in der Übung genannt wird, zum Beispiel `lab00`,
   `lab01` oder `lab02`.
4. Öffne die Node-Shells, die in der Übung genannt werden, zum Beispiel
   `lab03 - node1` oder `lab03 - router1`.
5. Folge den Hands-on-Übungen in
   [docs/hands-on-exercises.md](docs/hands-on-exercises.md).

### Referenzmaterial

- [docs/theory-slides-ipv4-networking.pdf](docs/theory-slides-ipv4-networking.pdf)
  enthält die Theorie-Slides, falls du später noch einmal etwas nachschauen
  möchtest.
- [lab-checkpoint-templates/](lab-checkpoint-templates/) enthält die
  Containerlab-Templates, mit denen die Lab-Checkpoints definiert werden. Diese
  Dateien brauchst du für das Tutorial nicht, aber sie sind interessant, wenn
  du neugierig bist, wie die Labs aufgebaut sind.

### Lab-Checkpoints

Die Lab-Gruppen `lab00` bis `lab06` sind Checkpoints. Jede Übung startet von
einem bekannten Lab-Zustand und führt zum nächsten Checkpoint. Wenn du nicht
weiterkommst, kann dein Tutor dich bitten, die aktuellen Node-Shells zu
schließen und eine spätere Checkpoint-Lab-Gruppe zu öffnen, damit du mit einem
sauberen Zustand weitermachen kannst.
